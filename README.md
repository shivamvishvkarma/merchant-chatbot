The AI-Powered Merchant Support Chatbot is a full-stack web application designed to assist merchants and developers with payment gateway integration,
onboarding, API documentation, and troubleshooting. 
It leverages Large Language Models (LLMs) to provide accurate, real-time responses, reducing manual support efforts and improving response times.

The application combines AI, Retrieval-Augmented Generation (RAG), and REST APIs to deliver context-aware answers from uploaded documentation,
making it an intelligent support assistant for payment and fintech platforms.
